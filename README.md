# 🚗 Araba Tutkunları

**Araba Tutkunları**, otomobil meraklıları için geliştirilmiş modern bir web platformudur.  
Sitede otomotiv haberleri, araç incelemeleri, karşılaştırmalar, tartışma alanları ve interaktif modifiye stüdyosu bulunmaktadır.

## 🔗 Proje Özeti

Bu proje, kullanıcıların otomotiv dünyasındaki güncel içerikleri takip edebileceği, araçları karşılaştırabileceği ve kendi araç modifiye yapılandırmalarını oluşturabileceği kapsamlı bir platform olarak geliştirilmiştir.

## ✨ Özellikler

- Modern ve responsive arayüz
- Türkçe / İngilizce dil desteği
- Kullanıcı kayıt ve giriş sistemi
- MongoDB bağlantılı kullanıcı yönetimi
- Giriş yapan kullanıcılar için admin paneli
- Haberler sayfası
- Araç incelemeleri
- Araç karşılaştırmaları
- Tartışma / yorum alanları
- CSS/3D tabanlı canlı modifiye önizleyici
- 43 parçalık modifiye yapılandırma sistemi
- Gerçek zamanlı fiyat hesaplama
- Hazır modifiye paketleri
- Build / yapılandırma özeti
- Yazdırılabilir yapılandırma çıktısı
- SEO uyumlu yapı

## 🛠️ Kullanılan Teknolojiler

- Next.js
- React
- TypeScript
- Tailwind CSS
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Lucide Icons
